### DeerFlow (bytedance/deer-flow)

**審計日期**：2026-04-06
**結論**：不安裝。設計精髓萃取為 E01-E08。
**一句話**：LangGraph 上的全棧 SuperAgent——功能最完整的開源 agent 框架，但預設安全配置為開發者體驗優化、非企業安全優化。

**基本資料**：

| 項目 | 值 |
|------|-----|
| Stars | ~57K (+9K/週，GitHub Trending #2) |
| 授權 | MIT |
| 語言 | Python (backend) + Next.js (frontend) |
| 核心依賴 | LangGraph, LangChain |
| 活躍度 | 1,833 commits (2025至今)，20+ 貢獻者 |
| 總分 | 7.5/10 |

**架構摘要**：
- Nginx → LangGraph Server (agent runtime) + Gateway API (REST) + Frontend (Next.js)
- 12 層 middleware chain：ThreadData → Uploads → Sandbox → DanglingToolCall → Guardrail → Summarization → TodoList → Title → Memory → ViewImage → SubagentLimit → Clarification
- Sandbox 雙模式：LocalSandbox（零隔離，開發用）/ AioSandbox（Docker 容器隔離）
- Subagent 系統：最多 3 並行、15min 超時、工具白名單
- 記憶系統：LLM fact extraction → debounced queue → JSON 持久化
- Guardrail：可插拔 provider、fail-closed、但預設關閉

**安全重點（決定不裝的主因）**：
- API 零認證 + CORS `*` = 非 localhost 即裸奔
- LocalSandbox 名為沙箱實為直接執行
- Bash 路徑驗證是 regex best-effort，shell 展開可繞
- Docker 映像預設從北京 Volcengine 拉取
- 前端 rehype-raw 無 DOM 消毒
- Docker socket 掛載 = 容器內可控 host

**正面觀察**：
- 無隱藏遙測/回傳（全面掃描確認）
- CLAUDE.md 556 行，是極優秀的 AI-readable 架構文件
- Harness/App 分層有 CI 測試守護
- 90+ 測試檔案

**使用者個人化配置（最終版）**：

- Project-level playwright fallback 保留（萬一 skillful 出問題）
