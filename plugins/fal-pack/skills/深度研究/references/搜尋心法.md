# 搜尋心法

> 讀取時機：測繪階段規劃偵察路線時載入。
> 體積上限：80 行。超過時觸發壓縮。

---

## 壓縮迴圈（維護規則）

每次偵察結束時：

1. **這次有非顯然的搜尋發現嗎？** — 大部分任務沒有，沒有就不寫。「Exa 搜得到結果」不是心法，「Exa 用 site: 限定後訊噪比提升三倍」才是
2. **跟已有心法有交集嗎？** — 有 → 濃縮合併，不是追加；沒有 → 單獨記錄，等待累積
3. **回測**：只靠這條心法，下次遇到類似場景能直接用嗎？不能 → 補上缺的脈絡
4. **超過 80 行？** → 強制壓縮：淘汰已被更好心法覆蓋的舊條目、合併重複的、刪過時的
