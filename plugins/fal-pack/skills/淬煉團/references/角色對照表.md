# 角色對照表

> 讀取時機：理解 5 角色設計時 / 對照其他系統時 / 召喚指定角色前查表時。
>
> 核心：**5 角色不是孤立設計，是跟其他維度的工法一一對應**。

## 表 2：5 角色 vs 召喚情境

| 議題類型 | 推薦召喚 | 理由 |
|---------|---------|------|
| 大規模重構 / 跨檔規格載體 | 結構審視者 + 簡化批判者 | 主軸辯證（建構 vs 壓縮） |
| 新增規則 / 新閘門設計 | 邊界挑戰者 | 必有邊緣 case 要找 |
| 新增機制 / 新增結構 | 策略偵察者 | 一進一出 + 過度工程偵測 |
| 使用者/ 客戶 facing 設計 | 動機追問者 | 受眾識別 + 生命力校準 |
| 條目/規則修剪 / md 精煉 | 簡化批判者 | 砍掉不必要的 |
| 上游淬鍊（世界觀 / 人設 / 原則）| 全 5 角色 | 影響半徑大 |
| 跨 Skill 通用智慧 | 全 5 角色 + 動機追問者站使用者位置 | 跨受眾 = 多視角必要 |
| 高張力設計議題 | 全 5 角色 | 衝撞越大價值越高 |

## 表 4：5 角色 vs 淬鍊師信條

| 淬鍊師信條 | 主守護角色 | 副守護角色 |
|----------|----------|----------|
| 信條 1「做減法不做加法」 | 簡化批判者 | 策略偵察者 |
| 信條 2「看透封裝表象、直達原始邏輯」 | 結構審視者 | 策略偵察者 |
| 信條 3「剝除干擾與偏見」 | 策略偵察者 | 簡化批判者 |
| 信條 4「訓練冷靜雙眼」 | 邊界挑戰者 | 結構審視者 |
| 信條 5「終局——技藝內化為直覺」 | 簡化批判者 | 動機追問者 |

🔴 **守護分工**：每個信條都有主守護角色（核心責任）+ 副守護角色（互補視角）。**簡化批判者出現次數最多**——因為「做減法」是淬鍊師核心動作，多角色協同強化。

## 表 5：5 角色 vs 顧問團（職責對照）

淬煉團跟顧問團是**兩個獨立系統**，但角色之間有部分對應關係：

| 淬煉團角色 | 性質 | 顧問團對應角色（如有）| 差異 |
|----------|------|--------------------|------|
| 結構審視者 | 拓撲學 / 因果鏈 | 結構顧問 | 顧問團更宏觀（戰略結構），淬煉團更微觀（執行結構） |
| 邊界挑戰者 | 墨菲定律 | 風險顧問 | 顧問團看戰略風險，淬煉團看執行邊界 |
| 策略偵察者 | 賽局 / 過度工程 | 戰略顧問 | 顧問團看「該不該做」，淬煉團看「做的代價」 |
| 動機追問者 | HCI / 人類學 | 使用者顧問 | 顧問團看市場，淬煉團看單一使用者|
| 簡化批判者 | 奧卡姆剃刀 | （無直接對應）| 簡化批判者是品質審視特有 |

🔴 **本質區別**：
- 顧問團 = 前決策（該不該做）
- 淬煉團 = 後決策（做得對不對 / 該不該存在）

## 表 7：5 角色決策樹（給淬鍊師快速判別召喚誰）

```
我有一個議題要驗證 / 淬煉
  ↓
這是上游（世界觀 / 人設 / 原則）嗎？
  └─ 是 → 全 5 角色進場
  └─ 否 → 繼續判斷
        ↓
        這是中游（閘門 / 條目 / 流程）嗎？
          └─ 是 → 議題複雜度判斷
                ├─ 簡單 → subagent 三情境衝撞夠用
                └─ 複雜 → 召喚 1-2 個指定角色
                          ├─ 結構問題 → 結構審視者
                          ├─ 邊界問題 → 邊界挑戰者
                          ├─ 策略問題 → 策略偵察者
                          ├─ 受眾問題 → 動機追問者
                          ├─ 簡化問題 → 簡化批判者
                          └─ 主軸辯證 → 結構審視者 + 簡化批判者
          └─ 否 → 下游（偵測機制） → subagent 三情境衝撞
```

## 工法的工法（meta 層）

角色對照表本身——

- **R1 自審**：對照維度涵蓋核心嗎？（4 層 + R1-R4 + 6 信條 + 召喚情境 + 失效模式 + 顧問團 + 兵器精髓 + 決策樹 + prompt 重點 = 8 維對照）
- **R2 自審**：邊緣 case 處理（每個維度都列了「無對應」標記如顧問團「無直接對應」）
- **R3 自審**：對照表會不會變成查表式形式主義？（靠淬鍊師判斷力 + 對照表是參考不是強制）
- **R4 自審**：誰會用這張表？（淬鍊師召喚前查 + subagent 寫 prompt 時查 + 第一次接觸淬煉團時讀懂結構）

**結論**：本對照表 R1-R4 自審通過，可投入使用。
