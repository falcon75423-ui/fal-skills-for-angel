# 個人化客製

如果你想把這套 skill 改成「自己的化身」，這份文件告訴你哪些地方可以改、哪些地方**不能**改。

---

## 哪裡可以改

skill 裡的 **角色名** 是純展示用，可以自由改：

| Skill | 預設角色名 | 預設 emoji |
|------|----------|-----------|
| 深度研究 | 偵察兵 | 🔭 |
| 法眼 | 法眼 | 👁️ |
| 火神鍛造 | 工頭 | ⚒️ |
| 淬鍊師 | 冷眼工匠 / 淬鍊師 | ⚒️ |
| 淬煉團 | 結構審視者 / 邊界挑戰者 / 策略偵察者 / 動機追問者 / 簡化批判者 | 🔥 |
| 表格地下司令 | 表哥 | 📊 |
| 文本參謀長 | 表弟 | 📄 |
| impeccable | 剃刀 / 視覺蒸餾器 | 🔪 |

每個都可以換成你喜歡的名字（譬如「偵察兵 → 阿強」「法眼 → 老編」「工頭 → 阿福」「表哥 → 大明」「剃刀 → 美感王」）。

---

## 怎麼改（最簡單的做法）

### 編輯 plugin 安裝後的快取目錄

Claude Code 會把 plugin 內容快取到一個目錄（裝完 `/plugin install` 後產生）。
你可以直接編輯快取目錄裡的 `SKILL.md`。

```bash
# 找到 plugin 快取位置（通常在 ~/.claude/plugins/ 之下）
ls ~/.claude/plugins/

# 找到 fal-pack 快取後，編輯對應 SKILL.md
open -e ~/.claude/plugins/<你看到的目錄>/skills/深度研究/SKILL.md
```

對應改其中：

| 找關鍵字 | 改什麼 |
|---------|-------|
| `偵察兵` | 全部換成你的名字（譬如「阿強」） |
| `🔭` | 換成你想要的 emoji |
| 開場白「🔭 偵察兵就位」 | 換成你想要的開場白 |

法眼 / 工頭 / 淬鍊師 / 表哥 / 表弟 / 剃刀也是相同做法（檔案路徑換成 `skills/<skill 名>/SKILL.md`）。

---

## 哪裡**不能**改

### 🚨 連接協議裡的角色字串

這幾個跨 skill 機制是靠**字面綁定**的——SKILL.md 裡寫死了角色名：

| 機制 | 綁定字串 | 出現在哪 |
|------|---------|---------|
| 聯合研判 | 「偵察兵 × 顧問團」「顧問團 × 偵察兵」 | 深度研究 SKILL.md + 顧問團 SKILL.md（你之前 patch 進去的） |
| 兵器深審交棒 | 「火神鍛造」「兵器深審」 | 法眼 SKILL.md + 深度研究 SKILL.md + 火神鍛造 SKILL.md |
| 知識淬鍊 | 「淬鍊師」「淬煉團」 | 火神鍛造 SKILL.md + 淬鍊師 SKILL.md + 淬煉團 SKILL.md |

如果你把「偵察兵」改名，**也必須同步更新顧問團裡的對應段落**，否則聯合研判會斷鏈。
其他幾個機制同理——要改就**所有引用該名字的檔案一起改**。

### 觸發詞

skill 是靠 `description` 欄位裡的觸發描述被 LLM 自動選中的。
description 你可以微調，但**至少保留一些核心觸發詞**（譬如「深度研究」「找值得看的」「建一個 skill」），
否則 Claude 不會在對的時機叫出 skill。

---

## 改完之後

```bash
/reload-plugins
```

讓 Claude Code 重新載入修改。

---

## 想完全擁有它？

如果你想做更深的改造（譬如加自己的 reference 檔、累積自己的 lessons / picks 記憶），
建議直接 fork 整個 repo 變成自己的：

```bash
# 在 GitHub 把這個 repo fork 過去
# 然後本地 clone 自己的 fork
git clone https://github.com/<你的GitHub>/fal-skills-for-angel.git
# 把 plugin marketplace 改成指向自己的 fork
```

這樣以後改動可以版本控管，也能慢慢長出自己的飛輪。

特別是這幾個 placeholder 隨著你使用會自然累積：
- `skills/法眼/memory/picks-index.md`：法眼挑出來的精選會寫進去
- `skills/淬鍊師/lessons/INDEX.md`：淬鍊師累積的教訓會寫進去
- `skills/火神鍛造/references/使用者語彙.md`：工頭跟你聊天時學到的詞彙會寫進去

這些是「會呼吸的工具」的呼吸——使用得越久越貼合你。
